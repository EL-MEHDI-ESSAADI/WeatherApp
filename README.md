# WeatherApp
This is a weather app, I build it using 
 - JS
    - Dom api
    - Xmlhttprequest api
    - Geolocation api
- CSS
  - SCSS Preprocessor
- HTML
## Used apis
- For reverse geocoding: openweathermap api (60req/1min)
- For get weather data: visualcrossing api (1000req/day)
